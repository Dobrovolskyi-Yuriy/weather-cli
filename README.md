# Weather CLI

## To start the project:
***1. Open folder with project in terminal and run***
```
npm i
```

***2. Create an account on the site and get API KEY***
```
https://openweathermap.org/
```

***3. Start project***
```
npm start -- -c [CITY] -t [API KEY]
```

***4. Help***
```
npm start -- -h
```
